# Миграция на новый домен suydacity.ru

## Пошаговая инструкция

### 1. Настройка домена в Vercel

1. Открой проект в Vercel: https://vercel.com/dashboard
2. Перейди в настройки проекта → **Settings** → **Domains**
3. Нажми **Add Domain**
4. Введи `suydacity.ru` и нажми **Add**
5. Vercel покажет DNS записи, которые нужно добавить:
   - **A record** или **CNAME record** для основного домена
   - Обычно это что-то вроде:
     - Type: `A`
     - Name: `@` (или пусто)
     - Value: IP адрес от Vercel (например, `76.76.21.21`)
     - ИЛИ
     - Type: `CNAME`
     - Name: `@` (или `www`)
     - Value: `cname.vercel-dns.com.`

### 2. Настройка DNS записей у регистратора домена

1. Открой панель управления доменом (где ты покупала suydacity.ru)
2. Перейди в раздел DNS / DNS Management
3. Добавь записи, которые показал Vercel:
   - Если Vercel показал A record:
     - Type: `A`
     - Host: `@` (или оставь пустым)
     - Value: IP адрес от Vercel
     - TTL: `3600` (или Auto)
   - Если Vercel показал CNAME:
     - Type: `CNAME`
     - Host: `@` (или `www`)
     - Value: `cname.vercel-dns.com.`
     - TTL: `3600` (или Auto)

4. Сохрани изменения
5. Подожди 5-60 минут (DNS может распространяться до 24 часов, но обычно быстрее)

### 3. Проверка DNS

После добавления DNS записей проверь:
- https://dnschecker.org/#A/suydacity.ru
- Должен показать правильный IP адрес от Vercel

### 4. Обновление email отправителя в Brevo (опционально)

Если хочешь использовать email с нового домена:
1. Открой Brevo: https://www.brevo.com/
2. Перейди в **Settings** → **Senders & IP**
3. Добавь новый sender:
   - Email: `noreply@suydacity.ru` (или другой email на твоем домене)
   - Name: `SYUDA`
4. Подтверди email (придет письмо на этот адрес)
5. После подтверждения обнови код в `api/auth/request-link.js`:
   - Замени `noreply@syuda.app` на `noreply@suydacity.ru`

### 5. Проверка работы

После настройки DNS:
1. Подожди 5-60 минут
2. Открой https://suydacity.ru
3. Проверь, что сайт открывается
4. Проверь авторизацию - отправь тестовое письмо
5. Проверь, что все функции работают

### 6. (Опционально) Редирект со старого домена

Если хочешь, чтобы старый домен перенаправлял на новый:
1. В Vercel в настройках старого домена (irkutsk.vercel.app) можно добавить редирект
2. Или оставь оба домена активными - они будут работать параллельно

## Важные моменты

- **DNS может распространяться до 24 часов**, но обычно работает за 5-60 минут
- **Все данные уже в Vercel KV** - они не зависят от домена, так что ничего переносить не нужно
- **Код автоматически определяет домен** из заголовков, так что изменения в коде не требуются
- **Email отправителя можно оставить как есть** (`noreply@syuda.app`) или обновить на новый домен

## Если что-то не работает

1. Проверь DNS записи через https://dnschecker.org/
2. Проверь логи в Vercel: **Deployments** → выбери последний деплой → **Functions** → смотри логи
3. Убедись, что домен добавлен в Vercel и показывает статус "Valid"
4. Проверь, что DNS записи сохранены у регистратора домена
