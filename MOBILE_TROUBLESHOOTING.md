# Решение проблемы с загрузкой на мобильных устройствах

## Проблема:
Сайт не загружается на мобильных устройствах (белый экран, даже API не отвечает).

## Возможные причины и решения:

### 1. Проблема с DNS на мобильном устройстве

**Решение:**
- На телефоне: Настройки → Wi-Fi → нажми на сеть → DNS → измени на `1.1.1.1` (Cloudflare) или `8.8.8.8` (Google)
- Или переключись на мобильный интернет (4G/LTE) и попробуй снова

### 2. Проблема с кешем браузера

**Решение:**
- **Safari (iPhone):** Настройки → Safari → Очистить историю и данные веб-сайтов
- **Chrome (Android):** Chrome → три точки → История → Очистить данные браузера
- Или открой в режиме инкогнито/приватном

### 3. Проблема с SSL сертификатом

**Решение:**
- Попробуй открыть `http://suydacity.ru` (без s) - если работает, проблема в SSL
- Проверь, что время на телефоне установлено правильно

### 4. Проблема с сетью/провайдером

**Решение:**
- Попробуй другую сеть Wi-Fi
- Попробуй мобильный интернет (4G/LTE)
- Попробуй VPN (если есть)

### 5. Проблема с браузером

**Решение:**
- Попробуй другой браузер (если Safari - попробуй Chrome, и наоборот)
- Обнови браузер до последней версии

## Диагностика:

### Проверка 1: DNS
На телефоне открой:
```
https://1.1.1.1
```
Если Cloudflare открывается - DNS работает.

### Проверка 2: Сеть
На телефоне открой:
```
https://google.com
```
Если Google открывается - интернет работает.

### Проверка 3: Наш сайт
На телефоне открой:
```
https://suydacity.ru/api/mobile-test
```
Должен вернуться JSON. Если нет - проблема с нашим сайтом.

## Если ничего не помогает:

1. Подожди 10-15 минут (возможно, DNS еще распространяется)
2. Перезагрузи телефон
3. Попробуй с другого устройства
4. Проверь, работает ли сайт с компьютера в той же сети Wi-Fi

## Контакт для поддержки:

Если проблема сохраняется, пришли:
- Какой браузер используешь (Safari/Chrome)
- Какой телефон (iPhone/Android)
- Что именно видно на экране
- Работает ли сайт с компьютера в той же сети
