# Настройка OpenAI API для рекомендаций

Для работы функции рекомендаций через ChatGPT необходимо настроить API ключ от OpenAI.

## Шаги настройки:

1. **Получите API ключ от OpenAI:**
   - Перейдите на https://platform.openai.com/api-keys
   - Войдите в свой аккаунт (или создайте новый)
   - Создайте новый API ключ

2. **Добавьте ключ в проект:**
   - Создайте файл `.env` в корне проекта (если его еще нет)
   - Добавьте строку:
     ```
     VITE_OPENAI_API_KEY=your_api_key_here
     ```
   - Замените `your_api_key_here` на ваш реальный API ключ

3. **Перезапустите dev-сервер:**
   ```bash
   npm run dev
   ```

## Важно:

- **Не коммитьте `.env` файл в Git!** Он уже должен быть в `.gitignore`
- API ключ хранится только на вашем компьютере
- Для продакшена используйте переменные окружения вашего хостинга (Vercel, Netlify и т.д.)

## Стоимость:

OpenAI API платный, но есть бесплатный кредит для новых пользователей. 
Модель `gpt-3.5-turbo` относительно недорогая (~$0.002 за 1000 токенов).

## Fallback режим:

Если API ключ не настроен, приложение будет использовать базовые рекомендации из локальной базы данных.
