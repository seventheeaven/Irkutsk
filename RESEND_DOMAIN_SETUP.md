# Настройка домена suydacity.ru в Resend

## Шаг 1: Добавление домена в Resend

1. Войди в свой аккаунт Resend: https://resend.com/login
2. Перейди в раздел **Domains** (Домены)
3. Нажми кнопку **Add Domain** (Добавить домен)
4. Введи `suydacity.ru` и нажми **Add**

## Шаг 2: Настройка DNS записей

Resend покажет тебе несколько DNS записей, которые нужно добавить в reg.ru:

### Типичные записи, которые нужно добавить:

1. **SPF запись** (TXT):
   - Имя: `@` (или `suydacity.ru`)
   - Значение: `v=spf1 include:resend.com ~all`

2. **DKIM записи** (TXT):
   - Resend покажет несколько записей вида:
     - Имя: `resend._domainkey` (или похожее)
     - Значение: `p=...` (длинная строка)

3. **DMARC запись** (TXT, опционально):
   - Имя: `_dmarc`
   - Значение: `v=DMARC1; p=none;`

## Шаг 3: Добавление записей в reg.ru

1. Зайди в reg.ru → Управление доменами → suydacity.ru → DNS-серверы и зона
2. Добавь все записи, которые показал Resend:
   - Тип: **TXT**
   - Имя: (то, что указал Resend)
   - Значение: (то, что указал Resend)
   - TTL: можно оставить по умолчанию

## Шаг 4: Ожидание проверки

1. После добавления записей вернись в Resend
2. Resend автоматически проверит DNS записи (может занять до 24 часов, обычно 5-15 минут)
3. Статус домена изменится с "Pending" на "Verified" (Проверен)

## Шаг 5: Обновление переменных окружения в Vercel

1. Перейди в Vercel → твой проект → Settings → Environment Variables
2. Убедись, что есть переменная `RESEND_API_KEY` с твоим API ключом Resend
3. Если её нет — добавь её
4. Перезапусти деплой (или он перезапустится автоматически)

## Важно:

- После того, как домен будет проверен в Resend, можно будет отправлять письма с адреса `noreply@suydacity.ru` на любую почту
- Пока домен не проверен, Resend может не отправлять письма или отправлять их только на твою почту
