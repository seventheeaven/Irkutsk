# Настройка Unsplash API для получения изображений Иркутска

## Шаги для получения API ключа:

1. Перейдите на https://unsplash.com/developers
2. Зарегистрируйтесь или войдите в аккаунт
3. Создайте новое приложение (Create a new application)
4. Заполните форму:
   - Название приложения (например, "Irkutsk Portal")
   - Описание (например, "City portal for Irkutsk")
   - Сайт приложения (можно указать localhost для разработки)
5. После создания приложения вы получите Access Key
6. Откройте файл `src/services/imageService.ts`
7. Замените `YOUR_UNSPLASH_ACCESS_KEY` на ваш ключ:

```typescript
const UNSPLASH_ACCESS_KEY = 'ваш_ключ_здесь';
```

## Лимиты Unsplash API:

- **Demo режим**: 50 запросов в час
- **Production режим**: После одобрения - 5000 запросов в час

## Альтернативные варианты:

Если вы не хотите использовать Unsplash API, можно:

1. **Pexels API** - также бесплатный, хорошая альтернатива
2. **Локальные изображения** - загрузить изображения в папку `public/img/`
3. **Другие сервисы** - Google Images API, Flickr API и т.д.

## Примечание:

В текущей реализации, если API ключ не установлен, будут использоваться fallback изображения (placeholder'ы).
